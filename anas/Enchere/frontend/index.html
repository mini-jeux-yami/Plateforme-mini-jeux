RÈGLES DU JEU

🎓 MANCHE D'ENTRAÎNEMENT

La 1re manche est gratuite — familiarisez-vous avec le système. Les points sont remis à zéro avant le vrai départ.

▸ SETUP

  • 3 estimateurs + 1 commissaire-priseur (MJ).
  • Le MJ voit la vraie valeur ; les joueurs ne voient que l'image.

▸ 3 ESSAIS PAR MANCHE

  • Chaque joueur a 3 essais pour estimer le prix.
  • Après chaque essai, une flèche indique si le prix réel est ⬆ plus élevé ou ⬇ plus bas.
  • Double flèche ⬆⬆ / ⬇⬇ si l'écart dépasse 50 %.
  • C'est le 3e essai qui est retenu pour le scoring.

▸ DÉROULEMENT

  1. Un objet apparaît à l'écran.
  2. Chaque joueur soumet ses estimations (3 essais max).
  3. Quand tous ont terminé, le MJ clique ⚡ ADJUGER.
  4. Estimations révélées → points distribués.
  5. On enchaîne jusqu'à épuisement des objets.

▸ SCORING (CRED)

  • 📏 PLUS PROCHE +1 pt au plus proche (ex-aequo : tous marquent).
  • ±5 % +1 pt si l'estimation est dans une marge de 5 %.
  • 🎯 EXACT +1 pt si l'estimation tombe pile sur la valeur.

→ Les 3 bonus se cumulent. Pile = 3 pts.

⚡ OVERCLOCK ROUND

Toutes les 10 manches réelles, les points sont doublés ×2.

▸ FIN DE PARTIE

Pile épuisée → le joueur avec le plus de CRED remporte la transmission.
